# LinkedIn Clone (Mini LinkedIn App)

A simple LinkedIn-style web application built with HTML, CSS, JavaScript, PHP, and MySQL.

## How to Run the Project Locally

1. Install XAMPP.
2. Copy this project folder into:
   C:/xampp/htdocs/
3. Create a database named linkedin_db in phpMyAdmin.
4. Import the file schema.sql from backend folder.
5. Update DB credentials in backend/db.php if needed.
6. Start Apache + MySQL in XAMPP.
7. Open:
   http://localhost/linkedin-mini-app/frontend/index.html

## Features
- Signup & Login
- Create Posts
- Public Feed
- User Sessions
- Logout
